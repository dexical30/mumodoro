
  # Pomodoro Music Timer App

  This is a code bundle for Pomodoro Music Timer App. The original project is available at https://www.figma.com/design/4b1cylYHCHxD0Y9XSIsDMU/Pomodoro-Music-Timer-App.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Environment variables

  To enable the YouTube search panel, create a `.env` file and provide a YouTube
  Data API key using Vite's `VITE_` prefix:

  ```bash
  VITE_YOUTUBE_API_KEY=your_api_key_here
  ```

  You can copy `.env.example` as a starting point.
